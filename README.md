# ObjAsm-C.2
Code base of the ObjAsm project.
New code is added here. A stable version can be downloaded from http://objasm.x10host.com/

## Installation

### Prerequisites
- Visual Studio 2017 or later (any edition)
- Windows 10/11 SDK

### Quick Start (recommended)

Run the interactive installation script:

```cmd
...\ObjAsm\Build\OA_INST.cmd
```

It verifies prerequisites, downloads UASM, lets you select a Visual Studio installation (if multiple are present), sets all required environment variables, and optionally compiles the libraries.

For detailed instructions see [`Help\Install\EN.adoc`](Help/Install/EN.adoc) (English) or [`Help\Install\DE.adoc`](Help/Install/DE.adoc) (German).

### WebInstaller (Alternative)

The WebInstaller downloads and extracts the ObjAsm package but does not run `OA_INST.cmd`. After using it, run `OA_INST.cmd` manually to complete the configuration:

[ObjAsm C.2.0 WebInstaller](https://objasm.x10host.com/DwnFiles/Ver_C.2.0/ObjAsm_C.2.0-WebInstaller.exe)

For detailed documentation, visit: http://objasm.x10host.com/
